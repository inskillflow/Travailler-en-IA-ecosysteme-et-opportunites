# Examen — Métiers Data, IA, Big Data, BI, GenAI & Marketing (version simplifiée)

**Durée :** 120 à 150 minutes

## Les métiers (rappels très courts)

- **BI Analyst / BI Developer** : construit des tableaux de bord lisibles pour la direction (Power BI, Tableau).  
- **Data Analyst** : analyse des données (SQL, Excel, pandas), calcule des indicateurs (KPI).  
- **Analytics Engineer** : prépare les tables prêtes pour l’analyse dans l’entrepôt de données (dbt).  
- **Data Engineer (Big Data)** : crée les tuyaux de données (ETL/ELT) et gère les gros volumes (Spark, Kafka, Airflow).  
- **Data Architect** : conçoit l’organisation des données (quelles tables, où, qui y accède).  
- **Data Steward / Gouvernance** : s’assure de la qualité, des définitions claires et des règles d’accès (PII).  
- **Data Scientist (ML “classique”)** : expérimente des modèles pour prédire (scikit-learn, stats).  
- **Applied Scientist / Deep Learning** : modèles profonds (images, texte, son) avec PyTorch/TensorFlow.  
- **ML Engineer** : met les modèles en service (API), rapide et fiable (FastAPI, Docker, K8s).  
- **MLOps / Platform** : automatise entraînement/déploiement/suivi (MLflow, Kubeflow).  
- **GenAI / LLM Developer** : construit des applis avec grands modèles de langue (prompts, RAG).  
- **Prompt Engineer** : écrit/structure les prompts et évalue la qualité des réponses.  
- **LLMOps / GenAI Platform** : mesure et contrôle les LLM en production (coûts, traces, règles).  
- **NLP Engineer** : traite du texte (classification, extraction), métriques F1/precision/recall.  
- **Computer Vision Engineer** : traite des images/vidéos (détection, segmentation).  
- **Generative Image Engineer** : génère des images (diffusion, sécurité du contenu).  
- **Marketing/Growth Analyst** : mesure l’impact des campagnes (cohortes, A/B test, CAC/LTV).  
- **Marketing Data Scientist** : modèles pour budget pub et attribution (MMM, uplift).  
- **AI Product Manager** : définit le problème, les métriques et la feuille de route produit.

> *Aides de vocabulaire (simples)*  
> - **Entrepôt de données (warehouse)** : grande base “officielle” pour l’analyse.  
> - **ETL/ELT** : déplacer/transformer des données d’une source vers une destination.  
> - **Kafka** : outil pour recevoir des données en continu.  
> - **RAG** : faire chercher des documents avant de répondre avec un LLM.  
> - **Drift** : quand les données changent avec le temps et que le modèle se dégrade.
